# 论文改进材料使用说明

本文件夹用于存放论文增补材料，不会修改你的 Word 文件。

## 文件结构

- `figures/`：建议插入论文的补充图片。
- `图片插入位置建议.md`：每张图建议放在哪一章、配什么图题、正文如何引出。
- `可复制增补内容.md`：可直接复制进论文的中文段落。
- `中文文献补充清单.md`：建议加入参考文献的法规、标准、论文和技术资料。
- `generate_figures.py`：生成本次图片的脚本，后续如果想微调图片文字可以改它再运行。

## 建议使用顺序

1. 先打开 `图片插入位置建议.md`，把 4 张图按章节放入论文。
2. 再从 `可复制增补内容.md` 中挑选段落，复制到对应章节。
3. 最后根据 `中文文献补充清单.md`，把适合的文献加入“参考文献”。

## 注意

部分法规、标准和论文资料需要你按学校参考文献格式再统一排版。这里先按“可读、可粘贴、可追溯”的方式整理。
